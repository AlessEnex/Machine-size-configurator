Overview

This guide helps new users use the configurator to obtain preliminary machine dimensions and 2D views.

Quick Flow

  1. Fill in the base information (Job, Cladding, Connections).
  2. Select tank data (Liquid Receiver and MT Suction Accumulator).
  3. Configure compressors and choose the Oil Separator.
  4. Select the B2B or Head electrical panel (you can use Magic Fill).
  5. Review the Top/Front views and apply a Manual Override if needed.
  6. Export the PDF with the desired configurations.

Required Sections

Sections with a yellow border indicate missing data needed to run calculations and visualization.

  • Tanks: Liquid Receiver (single or coupled) + MT Suction Accumulator.
  • Compressors: Oil Separator.
  • Panels: select at least one panel (B2B or Head).

Magic Fill (Panels)

The star button ✶ with the Magic Fill label automatically proposes panels based on the Liquid Receiver volume:

  • If volume ≤ 300 L → selects the largest panel with area ≤ 3 m².
  • If volume ≤ 700 L → selects the largest panel with area ≤ 5 m².
  • If volume > 700 L → selects the largest available panel.

The head panel is derived from the selected B2B:

  • B2B L800 → Head L1400
  • B2B L1000 → Head L1800
  • B2B L1200 → Head L2200
  • B2B L1400 → Head L2400

2D Views

Each configuration shows:

  • Top view: layout in plan (L/W).
  • Front view: side view with heights (H). The panel always has H = panel H + 150.

When cladding is active, it is shown as a single enclosure that includes the modules.

Manual Override

You can override the dimensions of each module (L/W/H). When Manual Override is active, a “Manual Override Active” badge appears.

Note: if you change the main options, overrides are automatically reset.

PDF Export

Select the configurations to include and generate the PDF. With Cladding = “No”, printable options for “Attached/Not Attached” are added.
